# Decode Labs CAD Drawing

## Project Overview
This repository contains the AutoCAD drawing file:

- `Decode Labs cop.dwg`

The drawing is intended for architectural/civil engineering drafting, detailing, documentation, and plotting workflows.

## File Information
- File Type: AutoCAD Drawing (`.dwg`)
- Main Drawing: `Decode Labs cop.dwg`

## Software Requirements
- AutoCAD 2018 or later (recommended)
- AutoCAD LT (viewing/editing)
- Autodesk DWG TrueView (viewing only)

## Drawing Standards
Recommended settings:
- Units: Millimeters (mm)
- Annotation Scale: As required by project
- Text Hierarchy:
  - Sheet Titles: 5–7 mm
  - Section Titles: 3.5–5 mm
  - Notes & Dimensions: 2.5–3 mm

## Contents
The drawing may include:
- Floor plans
- Dimensions
- Text annotations
- Layers
- Plot layouts
- Construction details

## Plotting
Before exporting to PDF:
1. Verify drawing scale.
2. Check layer visibility.
3. Confirm text readability.
4. Review dimension styles.
5. Use the appropriate CTB/STB plot style.
6. Export using PDF plotter settings.

## Quality Check
- Verify dimensions.
- Check lineweights.
- Confirm text hierarchy.
- Ensure layouts are correctly scaled.
- Review title blocks before submission.
