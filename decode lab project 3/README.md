Project 3: Commercial Office Space Layout
Overview

This project focuses on designing an efficient and functional corporate office layout using advanced AutoCAD referencing and
data management techniques. The objective is to create a professional workspace while demonstrating the use of External References (XREFs), Dynamic Blocks, and Attribute Extraction to streamline the design process and generate an automated Bill of Materials (BOM).

Project Objectives

Develop a well-organized commercial office interior layout.
Link the architectural shell drawing to the interior design using XREFs.
Create and utilize Dynamic Blocks for flexible furniture placement.
Assign attributes to furniture blocks for data extraction.
Generate an automated Bill of Materials (BOM) using AutoCAD Data Extraction tools.
Apply efficient space planning principles to maximize workspace functionality.
Features Implemented
1. External References (XREFs)
Architectural shell drawing attached as an external reference.
Interior layout maintained in a separate drawing file.
Automatic updating of referenced drawings when modifications are made.
2. Dynamic Blocks

Custom dynamic furniture blocks were created, including:

Workstation Desks
Workstation Chairs
Office Chairs
Meeting Table
Chairs
Office Tables
Sofas

Dynamic properties allow:

Stretching
Rotation
Easy modification without recreating blocks
3. Attribute-Based Furniture Management

Each furniture block contains attributes such as:

Furniture Type
Block Name
Price Identifier
Manufacturer

These attributes enable automated data extraction and inventory tracking.

4. Automated Bill of Materials (BOM)

Using AutoCAD Data Extraction:

Furniture data was extracted automatically.
Quantities were calculated and summarized.
Results exported to Microsoft Excel format.

5. Space Planning
The office layout includes:

Workstation Areas
Meeting/Conference Space
Reception Area
Waiting/Lounge Area
Circulation Paths for efficient movement

Workflow
Create the architectural shell drawing.
Attach the shell drawing as an XREF.
Develop dynamic furniture blocks with attributes.
Place furniture according to office space requirements.
Perform Data Extraction on attributed blocks.
Export extracted data to Excel.
Review and verify the generated BOM.
